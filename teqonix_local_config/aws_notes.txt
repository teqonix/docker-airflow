Need to create the following in Cloudformation:

VPC
    VPC Endpoints - https://docs.aws.amazon.com/AmazonECS/latest/developerguide/vpc-endpoints.html
        com.amazonaws.region.ecs-agent
        com.amazonaws.region.ecs-telemetry
        com.amazonaws.region.ecs
    VPC Security Group
        - Allow all traffic from VPC subnet 
        - Allow port 22 to EC2 instance
EC2:   
    - EC2 Launch Configuration
        - This seems to create the EC2 instance on its own..
        - AMI: Amazon ECS-Optimized Amazon Linux 2 AMI
    - EC2 Autoscaling Launch Configuration..
    - Add ec2_launch_script to User Data
        - Mount and format ECS volume (if it is brand new..)
    - Install the AWS ECS Container Agent https://docs.aws.amazon.com/AmazonECS/latest/developerguide/ecs-agent-install.html
    ECS Autoscaling Group 
        - Add EC2 
ECR Repository
ECS Task

EBS
    - 10GB gp2 volume

IAM
    - Need a role for EC2 instance 
        - AmazonEC2ContainerServiceforEC2Role
        - Access to EBS Volume
        - AutoScalingServiceRolePolicy (Part of the EC2 Autoscaling portion)
